# 玩家管理

"玩家"页展示服务器名册和最近活动，目前是以**查看**为主，部分管理操作按钮已经放在界面上但后端还没接入。

## 玩家名册

表格展示在线和离线玩家：

- **状态点**：在线（绿点）、等待加入中（黄点，包含正在连接/等待/加入几种子状态）、离线（灰点）。
- **在线时长/最近上线**：优先显示短格式时间点，比如"今天 14:20"、"昨天 09:10"、"3天前 20:00"；拿不到具体时间点才回退显示时长文案。
- **收入列**：玩家个人收入、农场总收入。
- 离线玩家如果存档文件里能找到但当前不在线名单，也会被列出并标注为离线，方便你确认这个存档一共有哪些玩家角色。

## 玩家活动 / 最近事件

分页展示的事件流，记录玩家加入、离开等动作，每页 2 条，用上一页/下一页翻页。

## 管理操作（部分待接入）

右侧"管理操作"卡列出四项：

| 操作 | 当前状态 |
| --- | --- |
| 踢出玩家 | 按钮已展示，**待接入**：需要 JunimoServer 提供对应 API 或可控命令后才能真正生效 |
| 封禁玩家 | 同上，待接入 |
| 白名单管理 | 同上，待接入 |
| 权限设置 | 同上，待接入 |

::: warning 目前能做的和不能做的
这四个按钮当前是禁用状态（灰色不可点），鼠标悬停会提示"XX 待接入"。如果需要临时踢人或限制某个玩家，目前只能通过 [服务器控制](/handbook/server-control) 页的控制台命令或直接重启服务器来间接处理，不存在面板内一键封禁/白名单的能力。这几项功能会随着 JunimoServer 上游能力完善后续跟进。
:::
