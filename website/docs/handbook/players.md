# 玩家管理

"玩家"页展示服务器名册和最近活动，支持踢出在线玩家和删除离线玩家角色；封禁、白名单、权限设置这几项管理操作按钮已经放在界面上但后端还没接入。

## 玩家名册

表格展示在线和离线玩家：

- **状态点**：在线（绿点）、等待加入中（黄点，包含正在连接/等待/加入几种子状态）、离线（灰点）、未知（灰点，状态无法读取）。
- **在线时长/最近上线**：优先显示短格式时间点，比如"今天 14:20"、"昨天 09:10"、"3天前 20:00"；拿不到具体时间点才回退显示时长文案。
- **收入列**：玩家个人收入、农场总收入（服务器运行中实时读取）。
- 离线玩家如果存档文件里能找到但当前不在线名单，也会被列出并标注为离线，方便你确认这个存档一共有哪些玩家角色。

## 玩家活动 / 最近事件

分页展示的事件流，记录玩家加入、离开等动作，每页 2 条，用上一页/下一页翻页。

## 管理操作

右侧"管理操作"卡列出四项：

| 操作 | 当前状态 |
| --- | --- |
| 踢出玩家 | **已接入**：可在玩家表格行内点踢出图标，或在下方选择在线玩家后点"踢出"，二次确认后提交 |
| 封禁玩家 | 按钮已展示，**待接入**：需要 JunimoServer 提供对应 API 或可控命令后才能真正生效 |
| 白名单管理 | 同上，待接入 |
| 权限设置 | 同上，待接入 |

::: tip 踢出玩家怎么用
只有当前**在线**且**不是主机**的玩家才能被踢出。点击踢出后会弹出二次确认框，确认后指令会提交给服务器内置的控制模组，通常几秒内生效，玩家会被断开连接（仍可以重新加入）。主机玩家出于安全考虑无法被踢出。
:::

::: tip 删除离线角色怎么用
如果某个玩家长期不来、占着角色位置，管理员可以在服务器**运行中**对该角色执行删除。操作入口在玩家名册的离线角色行，只对当前活动存档里已认领、非主机、且处于离线状态的角色生效。删除前面板会自动备份整档，删除后该角色和小屋将从存档中永久移除，玩家重新上线时会被自动中止连接。详见 [存档管理 - 删除离线玩家角色](/handbook/saves#删除离线玩家角色)。
:::

::: warning 目前还做不到的
封禁玩家、白名单管理、权限设置这三个按钮当前仍是禁用状态（灰色不可点），鼠标悬停会提示"XX 待接入"。如果需要限制某个玩家长期加入，目前只能通过设置[服务器加入密码](/handbook/server-control#服务器密码设置)间接控制，不存在面板内一键封禁/白名单的能力。这几项功能会随着 JunimoServer 上游能力完善后续跟进。
:::
