# 深度文档

这里按面板顶部导航的 9 个功能页逐页展开，讲清楚每个按钮、每个开关背后到底在做什么，适合部署完成后想吃透全部功能的用户。只想跑起来先玩，看 [快速上手](/guide/getting-started) 就够了。

::: tip 和《日常维护》的区别
[日常维护](/maintain/update) 是浓缩速查表；这里的深度文档是完整教程，覆盖每个页面的所有按钮、每种状态提示的含义，以及功能之间的联动关系。两边内容不冲突，深度文档遇到操作步骤时也会链接回日常维护页。
:::
