# 账号与权限

## 首次初始化

面板第一次访问（数据库里还没有任何用户）会自动进入管理员初始化页，设置用户名和密码。这个账号是第一个管理员，之后可以在"设置"页里再创建更多用户。

::: warning 密码要求
用**强密码**。面板本身没有暴露给公网的额外验证层，密码强度直接决定面板和 Docker 控制权的安全边界，参考 [面板设置 / 安全清单](/handbook/settings#安全维护清单)。
:::

## 登录与 Session

登录后面板通过 Session 维持登录态（`SETTINGS` 页"安全与权限"里的"Session 认证"项显示为已启用）。密码使用 bcrypt 强哈希存储，不会明文落库。

## 角色：管理员 / 普通用户

面板只有两种角色：

| 能力 | 管理员 | 普通用户 |
| --- | --- | --- |
| 登录、登出、查看自己信息 | ✅ | ✅ |
| 查看服务器状态、在线玩家、邀请码、存档/Mod 列表 | ✅ | ✅ |
| 全服喊话、执行只读控制台命令（服务器信息、邀请码等） | ✅ | ✅ |
| 导出 Mod 包/玩家同步包 | ✅ | ✅ |
| **启动 / 停止 / 重启服务器** | ✅ | ❌ 按钮不可点 |
| **踢出玩家、设置服务器加入密码** | ✅ | ❌ 按钮不可点 |
| **安装 / 上传 / 删除 Mod，一键安装 Nexus Mod** | ✅ | ❌ 按钮不可点 |
| **新建 / 切换 / 删除 / 上传 / 备份存档** | ✅ | ❌ 按钮不可点 |
| 用户管理（创建/禁用/删除账号） | ✅ | ❌ 看不到入口 |
| 查看 Docker 状态、诊断、审计日志 | ✅ | ❌ 看不到入口 |
| 配置 Nexus API Key、下载浏览器扩展 | ✅ | ❌ |

普通用户账号适合给一起玩的朋友：能查看服务器状态、在线玩家和喊话，但不能启停服务器、装 Mod、改存档或碰账号和底层运维，避免误操作影响正在运行的服务器。

## 用户管理

"设置"页"用户管理"区（仅管理员可见）可以：

- 创建新用户，指定角色（管理员/普通用户）。
- 禁用/启用某个账号。
- 删除用户：默认是**软删除**（禁用状态），需要显式勾选才会**硬删除**（数据库记录一起清除）。

::: warning 安全兜底
最后一个启用中的管理员账号不能被禁用、删除或降级为普通用户；当前登录的管理员也不能对自己执行禁用/删除。这是为了防止误操作导致面板无人能管理。
:::

## 审计日志

"设置"页"审计日志"区记录管理员的敏感操作（创建/删除用户、修改设置等），建议定期检查有没有陌生操作记录。
