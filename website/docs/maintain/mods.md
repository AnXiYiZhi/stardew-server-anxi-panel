# Mod 管理

"模组"页支持：

- 手动上传 Mod ZIP。
- 搜索 Nexus Mods 并通过浏览器扩展一键安装（含依赖识别：会提示缺失或未启用的前置 Mod）。
- 配置管理员的 Nexus API Key 后，Premium 用户可以走直连安装，无需浏览器扩展。
- 查看每个 Mod 的依赖是否满足、当前存档是否启用，按需启用/禁用（联动同包和前置 Mod）。
- 删除、导出已安装的 Mod。

::: tip
改动 Mod 后通常需要重启服务器才能生效，面板会在需要时提示"需要重启"。
:::

## Nexus 相关提示

- Nexus 数字 ID 精确查询需要先在"模组"页配置管理员的 Nexus API Key；未配置时会明确提示缺少 Key，而不是报错 500。普通关键词搜索不需要配置 Key。
- 通过浏览器扩展一键安装 Mod 时，确认扩展已更新到面板提示的最新版本（扩展版本和面板不匹配可能识别不到 Nexus 页面的下载入口）。
- 批量安装多个 Mod（含前置）时，如果某个 Mod 之前已经安装过又被重复提交，面板会自动识别为"已安装，跳过重复导入"，不会算作失败。
